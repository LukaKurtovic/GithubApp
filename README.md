# Requirements

Each implementation should contain following elements:

- MVVM architecture
- Androidx Lifecycle
- Jetpack Navigation Component
- Single Activity pattern
- Repository pattern
- Material design components
- Koin DI
- Retrofit and OkHttp
- Coroutines and flows
- 3rd party libs - coil/glide/picasso...
